A web interface for presenting bicycle count data from greenway infrastructure, tracking traffic and usage statistics on local greenway paths.

## About

As a board member of Bike Easy, a local bicycle advocacy group, I built this tool to help visualize cycling activity on New Orleans greenways. The display shows real-time and historical bike count data, helping demonstrate the value of cycling infrastructure to the community.

## Tech Stack

- Frontend: HTML and JavaScript
- Backend: Node.js server
- Deployment: Docker containerization
